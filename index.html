Cheatsheet

What is Bash

A command language interpreter.

It is widely available on various operating systems and is a default command interpreter on most GNU/Linux systems.

The name is an acronym for the ‘Bourne-Again SHell’.

What is Scripting?

Scripting allows for an automatic commands execution that would otherwise be executed interactively one-by-one.

Shebang or Hashbang

Tells the system that the commands that follow are to be executed by the Bourne shell which bash - to get location of bash 


							#!/bin/bash

First Script: Hello World

  • Create file named as hello.sh
  • Copy the following code: 
    
							#!/bin/bash
							
							echo "Hello World"
  • Make the script executable: chmod +x hello.sh
  • Running the script: ./hello.sh

VARIABLES

Can contain only letters (a to z or A to Z), numbers ( 0 to 9) or the underscore character( _). By convention, Unix shell variables will have their names in UPPERCASE.


							#!/bin/bash

							NAME="Ran"
							echo $NAME

Read-only VARIABLES

After a variable is marked read-only, its value cannot be changed.


							#!/bin/bash

							NAME="Ran"
							readonly NAME
							echo $NAME

Unsetting Variables

deleting a variable directs the shell to remove the variable from the list of variables that it tracks. Once you unset a variable, you cannot access the stored value in the variable.


								#!/bin/bash
	
								NAME="Ran"
								unset NAME
								echo $NAME
Command Line Arguments

a parameter that we can supply to our Bash script at execution.


							Command: ./multiple_argument.sh mango apple banana
						

							#!/bin/bash

							echo "The first index is": $0
							echo "The first fruit is: $1"
							echo "The second fruit is: $2"
							echo "The third fruit is: $3"
						

							Output

							The first index is: ./multiple_arguments.sh
							The first fruit is: mango
							The second fruit is: apple
							The third fruit is: banana

Reading all arguments and getting the length


							Command: ./multiple_argument.sh mango apple banana
						

							#!/bin/bash

							### Printing all arguments
							for TOKEN in $*
							do
								echo $TOKEN
							done

							### Getting the length of arguments
							echo 'Total Arguments:' $#
						

							Output

							mango
							apple
							banana
							Total Arguments: 3

Basic Operators


							#!/bin/bash

							read -p 'Please enter the first number:' FIRST
							read -p 'Please enter the second number:' SECOND
							read -p 'Please enter the operator:' OPERATOR

							### First Way 
							RES=`expr $FIRST $OPERATOR $SECOND` 
							echo "The result is: $RES"
							
							### Second Way
							RES2=$(($FIRST $OPERATOR $SECOND)) 
							echo "The result is: $RES2"

Reference for List of Operators: https://www.tutorialspoint.com/unix/unix-basic-operators.htm

Case Statements

						#!/bin/bash

						echo "Which color do you like best?"
						echo "1 - Blue"
						echo "2 - Red"
						echo "3 - Yellow"
						echo "4 - Green"
						echo "5 - Orange"
						read color;

						case $color in
							1) echo "Blue is a primary color.";;
							2) echo "Red is a primary color.";;
							3) echo "Yellow is a primary color.";;
							4) echo "Green is a secondary color.";;
							5) echo "Orange is a secondary color.";;
							*) echo "This color is not available. Please choose a different one.";; 
						esac
